React JS-კომპონენტები-დავალება 3

1. რა სახელი უნდა ერქვას მთავარ ჯავასკრიპტ ფაილს სადაც ყველაფრის თავმოყრა მოხდება და საბოლოო ჯამში რეაქტის აპლიკაცია რომელ ფაილში რენდერდება რათა საიტზე იყოს ნაჩვენები საბოლოო შედეგი?

მთავარ ჯავასკრიპტ ფაილი არის src ფაილში არსებული index.js ფაილი, საბოლოო ჯამში რეაქტის აპლიკაცია საიტზე რომ გამოვიდეს ის უნდა ჩარენდერდეს index.html ფაილში არსებულ მთავარ კომპონენტზე .

2. როგორი ტიპის კომპონენტების შექმნა შემიძლია რეაქტში?

რეაქტში შეგვიძლია შევქმნათ ორი ტიპის კომპონენტი: ფუნქციური და კლასის ტიპისა.

3. თუ მინდა კლასის ტიპის კომპონენტში ცვლადები გამოვიძახო, როგორ უნდა მოვიქცეტ ამისთვის?

ამისთვის აუცილებელია გამოვიყენოთ შემდეგი ფორმულა, კლასის ტიპის კომპონენტი უნდა იყოს შვილობილი რეაქტის კომპონენტის ასევე გვჭირდება კლასის ტიპის კომპონენტთან მუშაობისას render ფუნქცია. 

                class App extends React.Component { 
                    render() { 
                      return ( 
                        <React.Fragment>
                        <p>text </p>
                        </React.Fragment>
                      ) 
                    } 
                  }
